# Мои первые проекты на Python

Этот репозиторий содержит мои учебные проекты, которые я сделал во время изучения Python.

## Список проектов

### 1. 📝 Менеджер заметок (Tkinter)
Простое приложение для создания, редактирования и удаления заметок с категориями. Данные сохраняются в JSON-файл.

### 2. 💰 Финансовый трекер (Консоль + ООП)
Консольное приложение для учёта доходов и расходов. Хранит данные в JSON.

### 3. 📟 Калькулятор с историей (Tkinter + ООП)
Калькулятор с графическим интерфейсом, который сохраняет историю всех вычислений в JSON и позволяет просматривать её.

### 4. ❓ Игра "Верно или Нет?" (Tkinter)
Полноценная графическая игра, в которой нужно угадывать, верно ли решён математический пример. 
Есть настройки, квесты, статистика, объяснение как играть и сохранение в "невидимый" JSON-файл

### 5. ❌⭕ Крестики-нолики (Консоль + ООП)
Классическая игра, реализованная в консоли в объектно-ориентированном стиле. 
Есть обычный режим и песочница, где можно делать с игрой всё, что захочется

### 6. 📚 Менеджер книг (SQLite + ООП)
Консольное приложение для управления библиотекой книг. 
Использует ООП для организации кода и SQlite для хранения данных

### 7. 📄🔍 Текcтовый менеджер (Tkinter.filedialog + requests + ООП)
Консольное приложение, которое упрощает анализ текстовых файлов и интернет-страниц.
Можно посчитать слова, символы, посмотреть часто используемые слова, символы
и словосочетания, а также узнать, за сколько времени можно прочитать текст.

Приложение запрашивает открытие файла графически с помощью filedialog и
сохраняет историю обращений к файлам и страницам в JSON

Перед запуском данного проекта обязательно установите зависимости через requirements.txt

### 8. 🌦️ Погодный информер (SQLite + requests)
Консольное приложение для получения текущей погоды в любом городе мира через OpenWeatherMap API. Сохраняет историю запросов в SQLite. В программе есть объяснение, как получить API-ключ для запросов

Перед запуском данного проекта обязательно установите зависимости через requirements.txt
